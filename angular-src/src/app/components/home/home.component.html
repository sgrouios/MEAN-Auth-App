MEAN Authentication App

Welcome to my custom MEAN authentication application built from scratch

Register Login

Express.js Backend

A rock solid Node.js/Express.js server using Mongoose database
to organise models and query the database

Angular 11

Angular 11 framework to develop the front-end, using components, services, and guards - to name a few.

JWT Tokens

Full featured authentication using JSON web tokens.
Login and store user data

JWT silent refresh

Http interceptor configured to automatically generate
a new access token for the user when it expires.
No need for the user to login frequently.

Protected routes

Angular CanActivate guards implemented to
protect unauthenticated users from routes.
